## Penggunaan
- Clone repository ini
- Buat virtual env

pada OS berbasis Unix:

```python
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```
pada OS berbasis Windows:

```python
python -m venv env
env/Scripts/activate
pip install -r requirements.txt
```
- Jalankan

```python
python cli.py
```
